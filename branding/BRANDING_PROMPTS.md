# GitPulse - Visual Branding Prompts

**Tool:** GitPulse - Multi-Repository Health Monitor
**Style Guide:** Beacon HQ Visual System v1

## 1. TITLE CARD (16:9 Landscape - 1920x1080px)

**For:** GitHub repository header image, presentations, documentation

**DALL-E Prompt:**
```
Create a sleek, modern title card for "GitPulse" - a multi-repository health monitor tool.

STYLE:
- Dark navy blue background (#0a0e27)
- Bright cyan accents (#00d4ff)
- Tech-forward, professional aesthetic
- Minimalist geometric patterns

LAYOUT:
- Large bold title "GITPULSE" centered in modern sans-serif font
- Smaller tagline below: "Multi-Repository Health Monitor"
- Visual element: A grid of small repository icons (folder shapes) arranged in rows, each with a colored health indicator dot (green/yellow/red) showing different health states. A large magnifying glass or pulse/heartbeat line sweeps across the grid
- Subtle circuit board patterns and connected node lines in the background
- Small health grade badges (A, B, C) floating near some repo icons

MOOD: Professional, cutting-edge, reliable, monitoring dashboard feel, Team Brain ecosystem
```

## 2. LOGO MARK (1:1 Square - 1024x1024px)

**For:** Profile pictures, app icon base, social media

**DALL-E Prompt:**
```
Create a minimalist logo icon for "GitPulse" multi-repository monitor tool.

STYLE:
- Simple geometric shape combining git branch symbol with heartbeat/pulse line
- Bright cyan (#00d4ff) on dark navy (#0a0e27)
- Clean, recognizable at small sizes

DESIGN:
- Center: A git branch fork symbol (three dots connected by lines) with a heartbeat/ECG pulse wave running through it
- The pulse line passes through each node of the git branch, creating a visual fusion of "git" and "pulse"
- No text (icon only)
- Modern, professional, minimal
- Memorable silhouette that reads well at 32x32px

MOOD: Tech-forward, monitoring, health, reliability, part of unified Beacon HQ ecosystem
```

## 3. HORIZONTAL LOGO (3:1 Wide - 1800x600px)

**For:** Documentation headers, website banners, presentations

**DALL-E Prompt:**
```
Create a horizontal logo banner for "GitPulse" multi-repository monitor tool.

STYLE:
- Dark navy background (#0a0e27)
- Bright cyan text and accents (#00d4ff)
- Git branch + pulse icon on left side
- "GITPULSE" text in modern sans-serif font

LAYOUT:
- Icon (left 20%): Git branch symbol with heartbeat pulse running through nodes
- Tool name "GITPULSE" (center-left) in bold modern font, bright cyan
- Right side: A subtle visualization of multiple repository health indicators - small colored dots in a grid pattern representing repo health scores (mostly green/cyan with a few amber)
- Thin horizontal pulse line connecting the icon to the text

MOOD: Professional, cohesive branding, dashboard-like, Team Brain family
```

## 4. APP ICON (1:1 Square - 512x512px)

**For:** Desktop app icon, mobile app icon, favicon

**DALL-E Prompt:**
```
Create a simplified app icon for "GitPulse" multi-repository monitor tool.

REQUIREMENTS:
- Simple fusion of a git branch fork (3 connected dots) with an ECG heartbeat pulse wave
- Maximum 2 colors: cyan (#00d4ff) + white
- Dark background (#0a0e27)
- Bold, chunky shapes (readable at 32x32px)
- No text
- Flat design (no gradients)
- The heartbeat line should be the dominant visual element, with git branch nodes as accents

ICON STYLE: iOS/Material Design inspired, professional, instantly recognizable as both "git" and "health monitor"
```
